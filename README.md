# Bloom-Island-3D

Strengthen the island by gathering and developing.

## Game Description

To establish your own system on an island, gather wood and stone and progress through a simple development system. The game includes three different sources of income: **House**, **Lumber Yard**, and **Stone Statue**.

Players have a **Stamina Bar** and an **Experience Bar** that fills up as tasks are completed. There are **20 different tasks** available in the game.

With each level gained through experience:

* The time required to gather trees and stones decreases by **1 second**.
* The player receives a **+5 increase in resources**.

**House**
A random influx of **1 to 5 pieces of wood and stone every 5 seconds**.

**Lumber Yard**
Produces **20 logs of wood every 5 seconds**.

**Stone Statue**
Produces **20 stones every 5 seconds**.

## Features

* All tasks were implemented using **Scriptable Objects**.
* The **Singleton architectural design pattern** was used for core systems.
* A **building placement system** was implemented.
* Added **removable building controls**.
* Optimized sound effects using **AudioMixer**.
* Implemented character control using the **Unity New Input System**.
* Optimization techniques used:

  * Static Batching
  * Occlusion Culling
  * Texture Compression
  * Use of **4 Area Lights**
